# Supported Devices

MyTech-Fix can help troubleshoot virtually any consumer tech device. Coverage is deepest
for the most commonly used devices in each category.

## Categories with deep coverage

**WiFi & Networking** — routers, mesh systems (Eero, Netgear Orbi, TP-Link Deco, Google
Nest WiFi), modems, network switches, WiFi extenders

**Printers** — HP, Canon, Epson, Brother — connection issues, offline errors, print queue
problems, driver issues, wireless setup

**Smart Home** — Nest and Ecobee thermostats, Ring and Arlo cameras and doorbells, Philips
Hue lighting, SmartThings, Google Home, Amazon Alexa, Apple HomeKit, Matter devices,
Zigbee and Z-Wave devices

**Computers** — Windows 10/11, macOS, Chromebook — performance issues, WiFi problems,
update failures, blue screen errors, storage issues

**Security Cameras** — Ring, Arlo, Wyze, Google Nest Cam, Blink — offline cameras, video
feed issues, motion detection, night vision

**Phone & Tablet** — iPhone, iPad, Android — WiFi and Bluetooth connectivity, battery drain,
storage issues, sync problems

**VPN & Remote Work** — Windows and Mac VPN clients, remote desktop — connection failures,
speed issues, split tunneling

**Security & Privacy** — suspicious login alerts, router password hygiene, two-factor
authentication setup, WiFi security

## Cross-ecosystem conflicts

One area where MyTech-Fix is particularly strong is cross-ecosystem troubleshooting — when
two devices from different manufacturers won't work together. For example: Nest thermostat
not appearing in Google Home, Ring Doorbell not working with Alexa, or Philips Hue not
syncing with HomeKit. These issues are notoriously hard to search for and MyTech-Fix has
specific guidance for the most common combinations.
