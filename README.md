## Environment Variables

Copy `.env.example` to `.env.local` and fill the values (never commit `.env.local`).

Key ones for full functionality:
- Supabase (auth + DB)
- GROK_API_KEY (xAI for chat text)
- XAI_API_KEY (xAI for image generation in chat, uses grok-imagine-image-quality model)
- **OPENAI_API_KEY** (recommended for image generation — uses DALL·E 3 "hd" which produces much higher quality diagrams with far fewer spelling errors and better text legibility than xAI/Flux)
- Stripe keys (billing)

After changing `.env.local`, fully restart `npm run dev`.

See `.env.example` for details and comments.
